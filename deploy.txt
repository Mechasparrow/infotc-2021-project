Deploy steps

1. Install postgres https://www.postgresql.org/download/linux/ubuntu/
2. Install pyenv https://github.com/pyenv/pyenv-installer
3. 
Append following to .profile

export PYENV_ROOT="$HOME/.pyenv"
export PATH="$PYENV_ROOT/bin:$PATH"
eval "$(pyenv init --path)"

Append following .bashrc
eval "$(pyenv init -)"
eval "$(pyenv virtualenv-init -)"

source ~/.profile

4. pyenv install 3.6.5

5. Create deploy key
ssh-keygen -t ed25519 -C "your_email@example.com"
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519

6.
sudo apt-get install libpq-dev

7. To access postgres 
sudo -u postgres psql

CREATE DATABASE infotcdb;

7. git clone git@github.com:Mechasparrow/infotc-2021-project.git

8. In engineeringhubbackend
python manage.py migrate
python manage.py runserver 0.0.0.0:8000
